# Tribute Page – Steve Jobs

## 📌 Project Overview
This is a tribute page dedicated to Steve Jobs, created using HTML and CSS.
The page highlights his life, vision, and inspirational quotes with a clean
and minimal design.

## 🎯 Features
- Responsive layout
- Elegant black & white theme
- Biography section
- Inspirational quote
- Image integration

## 🛠️ Technologies Used

- CSS
