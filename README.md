## 🌟 About The Repository

A curated collection of **frontend development projects** built using **HTML, CSS, and JavaScript**.

This repository contains beginner-to-intermediate level projects focused on:

✨ Responsive UI Design  
✨ Interactive Components  
✨ CSS Animations  
✨ JavaScript DOM Manipulation  
✨ Real-world Frontend Concepts  

Whether you're learning frontend development or exploring creative UI ideas, this repository serves as a practical showcase of modern frontend fundamentals.

# ⚙️ Getting Started

## 📥 Clone The Repository

```bash
git clone https://github.com/Mantechie/Frontend_Projects.git
```

## 📂 Navigate Into The Project

```bash
cd Frontend_Projects
```

## ▶️ Run Any Project

Open any project folder and launch the HTML file in your browser.

Example:

```bash
cd Weather
```

Then open:

```bash
weather.html
```
